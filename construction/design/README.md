# Design Documents - VT PhD Qualifier

This folder contains design documents for the PhD qualifier preparation process.

## Documents

| Document | Status | Description |
|----------|--------|-------------|
| [qualifier-design.md](qualifier-design.md) | Complete | Comprehensive design for the IEEE conference paper and oral presentation. Covers paper architecture, section-by-section content plan, thematic mapping strategy, research proposal design, presentation structure, writing workflow, and quality gates. |
| [slide-polish-design.md](slide-polish-design.md) | Complete | Design for polishing slides 2-20 of the Beamer presentation: inserting speaker notes from `files/slides.md`, tightening body text, and adding or improving TikZ/image visuals on every slide. |

## Document Naming Convention

Use descriptive names that indicate the document focus:
- `{topic}-design.md` - Design and specification documents
- `{topic}-analysis.md` - Analysis documents
- `{area}-study-plan.md` - Study plan documents
