# Requirements - VT PhD Qualifier

This folder contains requirements and acceptance criteria for the PhD qualifier preparation.

## Documents

Requirements documents will be added as preparation progresses:
- Exam format and requirements
- Preparation checklists
- Quality standards for study materials

## Quality Standards

### PhD Qualifier Requirements

- Demonstrate mastery of core subject areas
- Show ability to apply concepts to research problems
- Clear and rigorous technical communication
- Breadth across required topics
- Depth in specialization area

### Preparation Standards

- Every core topic covered with study notes
- Practice problems completed for each area
- Key papers and references reviewed
- Mock exam(s) completed

## Document Naming Convention

Use descriptive names:
- `{topic}-checklist.md` - Verification checklists
- `{area}-requirements.md` - Area-specific standards
- `{quality}-criteria.md` - Quality acceptance criteria
