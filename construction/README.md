# Construction Folder - VT PhD Qualifier

## Purpose

This folder tracks the current state of PhD qualifier preparation and development work. It should be updated **before and after each work session**.

## Project Context

**Project:** VT PhD Qualifying Exam Preparation
**Goal:** Prepare and pass the Virginia Tech PhD qualifying examination
**Status:** Project initialization

## Structure

### design/
Contains design documents for the qualifier preparation:
- Research area analysis
- Study plan documents
- Topic coverage mapping

### requirements/
Contains quality requirements and acceptance criteria:
- Exam requirements and format
- Preparation checklists
- Quality standards for deliverables

### sprints/
Contains sprint documents for preparation work:
- Current sprint status
- Completed tasks
- Remaining work

## Workflow

1. **Before working:** Review current sprint and active context
2. **When preparing a topic:** Check requirements/ for quality standards
3. **During work:** Update progress in sprints/ folder
4. **After session:** Update memory-bank and verify progress
